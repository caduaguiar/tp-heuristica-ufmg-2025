# Introdução

Markdown para iniciarmos a escrita do TP - 1 (Introdução) a ser entregue até o dia 28/09/2025, às 23:59, em formato PDF. A entrega deve seguir o roteiro disponibilizado em: https://docs.google.com/document/d/1aH-Ywm-Hkucb6Jfay7CDo92YzxU6eC7LyVx4ff3gBnI/edit?tab=t.0

Os seguintes itens são contemplados:

1) Contextualização (do problema real)
2) Definição do problema (de otimização)
3) Caracterização do problema (Classe de complexidade)
4) Caracterização do problema (Problemas semelhantes, caso hajam)
5) Caracterização do problema (Qualquer outra informação relevante)
6) Motivação (Por que vale a pena estudar este problema)
7) Objetivos (o que você pretende desenvolver nesta pesquisa)
8) Justificativa (Por que é importante atingir esses objetivos)
9) Roteiro (O que esperar nas próximas seções)

No roteiro, o professor deixa claro que ele deve seguir exatamente a estrutura acima, onde cada subtópico seja composto preferencialmente (mas não obrigatoriamente) por um parágrafo. Como são 09 tópicos, podemos fazer a seguinte divisão:
- Integrante 01: 01-03
- Integrante 02: 04-06
- Integrante 03: 07-09
